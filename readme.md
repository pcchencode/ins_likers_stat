## `ig_script`產出按讚用戶統計，統計每個按讚用戶的總讚數，找出您的頭號粉絲！

### 參數說明
*	`target_id`: 輸入您想要查找的`ins account`
*	`user_ac`: 輸入您所要登入的帳號
*	`user_pw`: 輸入您所登入帳號的密碼

### 套件需求
* `pandas`
* `selenium`
* `chromedriver.exe`

### 執行方法
* `python3 ig_script.py -targer_id YOUR_TARGET_ACCT -user_ac YOUR_LOGIN_ACCT -user_pw YOUR_PASSWORD`
